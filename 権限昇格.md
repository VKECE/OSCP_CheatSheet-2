# Linux 権限昇格

### 基本チェック
`whoami`  
`id`  
`hostname`
`cat /etc/issue`  
`cat /etc/*-release`  
`cat /etc/passwd`  

`ps -aux`  
`ip -a`  
`netstat -ano`  
`/sbin/route`  
`ss -anp`  

`ls -lah /etc/cron*`  
`cat /etc/crontab`
`grep "CRON" /var/log/cron.log`

`dpkg -l`  
`find / -writable -type d 2>/dev/null`

`cat /etc/fstab`  
`mount`  

`/bin/lsblk`  
`lsmod`  
`/sbin/modinfo <module>`  

`find / -perm -u=s -type f 2>/dev/null`
